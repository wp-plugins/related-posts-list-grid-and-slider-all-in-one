Features

  • Display Related Posts automatically after the content on your website without the need to edit template files

  • Relevancy matching based on title and/or tags of the post

  • Very easy to use options page

  • Custom options for each mode so you can shift between modes without need to change values every time!

  • Contain three amazing modes to display related items as (list, grid or slider).

  • Ability to show related posts as a list of rows

  • Ability to show related posts as a grid of items

  • Ability to show related posts as a slider at the bottom of the current post

  • Ability to control the sizes of the thumps as you like

  • Ability to show/hide thumps as you like

  • CSS3 some effects are supported (working fine with newer version of browsers)

  • Support for WordPress post thumbnails new feature

  • You can use images from custom image fields (image, featured, photo or etc..) if there is no featured image set to the post

  • You can control the number of related posts will be shown after the post content

  • You can control the title text of the related posts block

  • Space between items and rows can be controlled also

  • Display excerpts in post. Excerpt length in words can be controlled in the list mode

  • Output fading in the slider mode using jQuery after scrolling the page
